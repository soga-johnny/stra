## 1. 概念の位置づけ

### プラズミズムとは

プラズミズム（Plasmism）は、**主体が自己の差分構造と関係する能力の状態**を記述するモデルであり、実行基盤（豊かさ条件⑤）の内的次元における適応度の指標として機能する。

Straの存在論は「自己 = 差分の総体」として主体を**外部から構造的に**記述する。しかしその記述は評価圏との相対において定義されたものであり、主体の外側に位置する。この外部定義された自己像を個人として内側から引き受け、豊かさへ向けて実装するためには、主体自身が差分を認識・扱い・設計・実行できる内的能力を持つことが必要である。

プラズミズムはその**内的能力の状態と遷移**を記述する。

```
【外部・構造的記述】Straの存在論
  評価圏 → 差分の観測 → 自己 = 差分の総体

           ↓ しかし、これは外側から定義された自己像

【問い】その自己像を個人として内側から引き受け、豊かになるためには？

           ↓

【内部・実装的記述】プラズミズム
  差分と適切に関係できる内的状態 = 実行基盤の内的次元
```

### 体系内の位置

| 層 | 性質 | プラズミズムの役割 |
|---|---|---|
| **構造層** | 価値中立 | 差分との関係能力の4状態を記述する。状態に本質的な善悪を帰属させない |
| **目的層** | 規範的 | 豊かさの実現可能性を高めるための内的基盤の指標として機能する |

豊かさの5条件（`foundations/00-principles.md`）との関係：

- 条件⑤「実行基盤」は従来、**外的次元**（資源・能力・アクセス権）として定義されている
- プラズミズムは条件⑤の**内的次元**（差分との関係能力の状態）として機能し、外的次元を補完する

> → 豊かさの5条件との対応: [§ 6](#6-豊かさの5条件との対応)

## 2. 核心命題

> 評価圏によって定義された差分の総体としての自己を、個人として豊かに生きるためには、  
> その差分構造に相応に関与できる内的状態が必要である。

評価圏は「主体が何者であるか」を外側から定義する。  
プラズミズムはその外部定義された自己像を「内側から引き受けられるかどうか」の状態と遷移を記述する。

---

## 3. 物質の4状態との対応

プラズミズムは物質の4状態（固体・液体・気体・プラズマ）を比喩として採用する。  
物理的な状態変化を起こすトリガーが**熱量（エネルギーの付加）**であるように、  
プラズミズムにおける状態遷移のトリガーは**差分統合量**である。

### 差分統合量（Deviation Integration Volume）

> 評価圏との接触 → 差分フィードバックの受容 → 中央値の更新 → 差分の再設計  
> というサイクルの累積深度

| 物理的対応 | Straにおける対応 |
|---|---|
| 熱量（エネルギーの付加） | 差分統合量（評価圏との接触サイクルの累積） |
| 分子の平均運動エネルギー | 差分との関与の深度 |
| 外部熱源 | 評価圏（外部フィードバックの発生源） |

差分統合量はStraのサイバネティックループ（中央値の仮置き → 外部フィードバック → 補正）が活性化された頻度と深度の積分値に相当する。入力差分の**設計**と出力差分の**観測・実行**の両方を包含する概念である。

## 4. 4つの適応状態

### 固体（Solid State）

**定義**：自己像が硬直・固定化し、差分を認識・受容できない状態。

| 特性 | 内容 |
|---|---|
| 中央値への関係 | 中央値と差分を区別できない。自己を固定実体として認識する |
| 外部フィードバック | 拒絶・無視する。外部評価を脅威として処理する |
| 差分設計 | 差分の存在を前提として設計できない |
| 評価圏への関係 | 認識できない、または単一の評価圏にのみ固着している |

**物理比喩**：分子が固定格子に束縛され、外力に対して形を変えることを拒む。内部に整然とした秩序を持つが、外部の変化に応答できない。

---

### 液体（Liquid State）

**定義**：複数の所属・評価圏を認識し、文脈に応じて自己像を変容できる状態。

| 特性 | 内容 |
|---|---|
| 中央値への関係 | 差分の存在に気づいている。複数の中央値を認識できる |
| 外部フィードバック | 受容できる。ただし処理・統合には時間がかかる |
| 差分設計 | 差分の認識はできるが、意図的な設計はまだ難しい |
| 評価圏への関係 | 複数の評価圏を認識・切り替えられる |

**物理比喩**：容器（評価圏・文脈）の形に合わせて変化できるが、体積（核となる自己認識）は保持する。形は変わるが量は変わらない。

---

### 気体（Gaseous State）

**定義**：あらゆる評価圏・文脈に浸透でき、差分を俯瞰的に観察・設計の対象として扱える状態。

| 特性 | 内容 |
|---|---|
| 中央値への関係 | 複数の中央値を同時に把握し、自己の差分構造を俯瞰できる |
| 外部フィードバック | 統合できる。中央値の更新に活用できる |
| 差分設計 | 入力差分の設計が可能。ただし実行・フィードバックループの自律的運用はまだ不安定 |
| 評価圏への関係 | 所属・評価圏の境界にとらわれず複数を渡り歩ける |

**物理比喩**：境界を持たず空間（評価圏）全体に自由に拡散・浸透できる。どの容器にも入ることができ、形にとらわれない。

---

### プラズマ（Plasma State）

**定義**：差分に高感度で反応し、評価圏との相互作用の中で差分を能動的に設計・実行・最適化できる状態。

| 特性 | 内容 |
|---|---|
| 中央値への関係 | 中央値の変化を即座に感知し、差分構造を動的に再記述できる |
| 外部フィードバック | 即時に統合し、入力差分の再設計に反映できる |
| 差分設計と実行 | 設計→実行→出力差分観測→補正のサイクルを自律的に回せる |
| 評価圏への関係 | 評価圏との相互作用を通じて、最適な差分形状を能動的に生成できる |

**物理比喩**：電子が原子核から解放され、電磁場（評価圏）と最大限に相互作用できる最高エネルギー状態。外部場との相互作用によって形を成形しながら、同時にその場に影響を与える。

## 5. 状態遷移モデル

```
固体 ──[差分統合量の増加]──> 液体 ──> 気体 ──> プラズマ
固体 <──[差分統合量の停止・喪失]── 液体 <── 気体 <── プラズマ
```

### 各遷移の臨界点

| 遷移 | 内容 |
|---|---|
| 固体 → 液体 | 最初の外部フィードバックの受容。複数の所属・評価圏の認識の開始 |
| 液体 → 気体 | 差分を「設計の対象」として捉える視点の獲得。複数評価圏での差分経験の蓄積 |
| 気体 → プラズマ | 入力差分→評価圏→出力差分の設計・実行サイクルを自律的に回せるようになること |

### 退行（降温）

状態は後退しうる。プラズマ状態にある主体も、以下の要因により液体・固体状態へと退行することがある。

- 評価圏からの長期的な遮断・孤立
- 外部フィードバックの継続的な拒絶
- アイデンティティへの強い脅威（強度の否定・批判）
- 固定的な自己像への意図的な回帰

退行はStraの文脈では**差分統合量の停止または喪失**として記述される。

## 6. 豊かさの5条件との対応

| 適応状態 | 成立している豊かさの条件 |
|---|---|
| 固体 | 条件①が未成立、または表面的にのみ成立 |
| 液体 | 条件①②が成立（所属認識・認識の更新可能性） |
| 気体 | 条件①②③が成立（選択肢の存在まで） |
| プラズマ | 条件①〜⑤が同時成立 ＝ 豊かさの状態 |

プラズミズムの状態は豊かさの5条件の**実装可能性の指標**として機能する。  
条件が表面上成立していても、適応状態が相応でない場合、豊かさは構造的に実現されない。

> → 豊かさの5条件の定義: [foundations/00-principles.md § 3](00-principles.md)

## 7. 設計と実装の統合

プラズミズムは「記述」にとどまらず、**現在状態の診断**と**次状態への設計・実行**を統合する。

### 現在状態の診断

以下の問いにより、現在の適応状態を推定できる。

| 診断問い | 固体 | 液体 | 気体 | プラズマ |
|---|---|---|---|---|
| 外部フィードバックを受け取れるか | 難しい | できる | できる | 即時統合 |
| 複数の評価圏を認識できるか | できない | できる | 自在 | 能動的に選択 |
| 差分を設計の対象として扱えるか | できない | 難しい | できる | 自律的に設計・実行 |
| 入力差分→出力差分のサイクルを回せるか | できない | できない | 不安定 | 自律的に回せる |

### 各状態からの遷移実践

**固体 → 液体**
- 単一の評価圏から出て、異なる評価圏との接触を試みる
- 外部フィードバックを小さな単位から受容する練習
- 「自分は複数の文脈に属している」という認識を実験的に試みる

**液体 → 気体**
- 複数の評価圏において自分の差分を意識的に観察・記述する
- 差分を「固有の本質」ではなく「相対的な位置」として捉え直す
- 入力差分の最初の設計を試みる（→ [practice/01-deviation-design.md](../practice/01-deviation-design.md)）

**気体 → プラズマ**
- 設計した入力差分を実際に実行し、出力差分を観測する
- 補正ループを意識的に回す（設計 → 実行 → 観測 → 補正 → 再設計）
- 評価圏を能動的に選択・変更しながら差分の最適化を試みる

## 8. Straの既存概念との接続マップ

| Straの概念 | プラズミズムとの接続 |
|---|---|
| 差分（Deviation） | 各状態が「差分と関係する能力の深度」を記述する |
| 評価圏（Evaluation Sphere） | 差分統合量のトリガーとなる接触相手。液体以上の状態で複数認識が可能になる |
| 中央値（Median） | 固体では認識不可。プラズマでは変化の即時感知・再設計が可能 |
| 豊かさの5条件 | 各状態が条件①〜⑤の成立度と対応する |
| 実行基盤（条件⑤） | プラズミズムは条件⑤の内的次元として機能し、外的次元（資源・アクセス権）を補完する |
| 入力差分・出力差分 | 気体以上の状態で設計可能。プラズマで自律的サイクルとして実装可能 |
| サイバネティックループ | 差分統合量の正体。このループの累積がプラズミズムの遷移を駆動する |

## 9. 組織主体への拡張（仮定義）

> ⚠️ 本節は**仮定義段階**の記述である。個人（ヒト主体）のプラズミズムを組織主体へ類推的に拡張したモデルであり、今後の精緻化・検証が必要。

### 前提

プラズミズムは個人の心理的・認知的な内的状態として定義されている。  
組織（法人・団体・チーム）も主体として豊かさの5条件が適用されるが、その内的次元は心理的状態ではなく**組織の構造・学習・文化に宿る適応能力**として現れる。

```
【個人主体】
  外部次元: 差分 → 評価圏 → 資本変換
  内部次元: プラズミズム（心理的・認知的適応状態）

【組織主体】
  外部次元: 差分 → 評価圏 → 資本変換（同一メカニズム）
  内部次元: 組織的適応状態（学習・文化・ガバナンスに宿る適応能力）
```

### 豊かさの5条件の組織的形態

| 豊かさの条件 | 個人の形態 | 組織の形態（仮） |
|---|---|---|
| ① 所属の認識 | 自身の帰属文脈の自覚 | 参入市場・産業・評価圏の組織的認識 |
| ② 認識の更新可能性 | 自己像の外部補正 | 組織学習・市場フィードバックの内部反映経路 |
| ③ 選択肢の存在 | 複数評価圏へのアクセス | 複数市場・資本変換経路の確保 |
| ④ 意志の保持 | 自律的選択の維持 | ガバナンス・意思決定主体性の維持 |
| ⑤ 実行基盤 | 資源・能力・内的適応状態 | 財務基盤・組織能力・組織文化（内的次元） |

### 組織の4つの適応状態（仮）

| 状態 | 組織の様態 | 主な特性 |
|---|---|---|
| **固体（組織的固体）** | 硬直型 | 外部フィードバックを制度的に排除。単一評価圏に固着。「これまでのやり方」から動かない |
| **液体（組織的液体）** | 改善型 | 市場フィードバックを受容できる。複数市場を認識できるが、差分設計はまだ戦略的でない |
| **気体（組織的気体）** | 戦略型 | 差分を意図的に設計し、複数評価圏を横断できる。組織学習をフィードバックに活用できる |
| **プラズマ（組織的プラズマ）** | 自律進化型 | 差分設計→実行→補正サイクルが組織として自律的に機能。評価圏を自ら変容させる力を持つ |

### 状態遷移のトリガー（組織版・仮）

個人の差分統合量に相当するのは、組織における**評価圏との接触サイクルの累積**である。  
市場フィードバックの受容・組織学習の深度・ガバナンス更新の頻度が積み重なることで状態が遷移する。

```
固体 ──[外部フィードバック受容の制度化]──> 液体 ──> 気体 ──> プラズマ
固体 <──[組織の硬直化・外部遮断]── 液体 <── 気体 <── プラズマ
```

> → 組織形態別の資本構成傾向と適応状態の対応: [cases/01-capital-balance-patterns.md](../cases/01-capital-balance-patterns.md)
