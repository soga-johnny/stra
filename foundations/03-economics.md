## 1. 差分だけでは資本は生まれない

差分が存在することと、差分が社会的・経済的価値を持つことは別である。

```
Difference ≠ Value ≠ Capital

例: 世界一変わった石 → 差分はある → 評価する圏がなければ価値はゼロ
```

差分が資本へ変換されるには以下のプロセスが必要:

```
差分 → 認識 → 充足ニーズ（Job）との接続 → 評価圏 → 資本化 → 再生産
```

---

## 2. 評価圏（Evaluation Sphere）

### 定義

評価圏とは、**差分が評価され資本へ変換される社会的環境**である。

> ⚠️ 評価圏は「所属」とは異なる概念である。  
> 所属は主体が帰属するセグメントだが、評価圏は差分が評価される場・文脈である。  
> 主体は所属を持つが、評価圏は主体の外側に存在する。

### 評価圏の例

| 評価圏 | 評価される差分の性質 |
|---|---|
| 市場 | 経済的有用性・収益可能性 |
| 文化圏・芸術界 | 文化的独自性・美的差異 |
| 学術界 | 知識の新規性・論理的厳密性 |
| SNS / メディア | 認知・共感・拡散可能性 |
| スタートアップ界 | 問題解決力・スケーラビリティ |
| コミュニティ | 関係性・信頼・共通価値との差異 |

### 評価圏と価値の可変性

同一の差分でも評価圏が異なれば価値は変化する。

```
例: 過激なアート作品
  芸術界    → 高い文化資本
  一般市場  → 低い経済資本
  SNS      → 高い認知資本
```

## 3. 資本への変換（ブルデュー資本理論との接続）

評価圏において認識された差分は資本へ変換される。  
この資本はブルデューの資本理論の3分類と対応する。

| 資本の種類 | 定義 | Stra上の意味 |
|---|---|---|
| **経済資本** | 金銭・財・生産手段 | 市場評価された差分 |
| **社会関係資本** | 人脈・ネットワーク・信用 | ネットワーク評価された差分 |
| **文化資本** | 知識・技能・学歴・文化的嗜好 | 文化的評価を受けた差分 |

```
Capital = Difference × EvaluationSphere × Time
```

資本の種類は評価圏によって決まる。同じ差分が複数の評価圏で異なる種類の資本に変換されうる。

---

## 4. 差分が価値になる3条件

資本化の前提として、差分が評価圏内で価値を持つには以下の3条件が必要。

### ① 認識可能性
差分は他者が認識できる必要がある。  
認識されない差分は評価圏に入れない。

> 例: 技術・デザイン・思想 → 可視化されて初めて評価される

### ② 希少性（Scarcity）
差分は再現困難である必要がある。  
模倣されやすい差分は資本としての価値を失う。

> 例: アート・ブランド・専門知識 → 模倣コストが高い

### ③ 充足ニーズ（Job）との接続
差分が他者の充足ニーズ（Job）と接続される必要がある。  
充足ニーズとは「その人・組織が果たしたい役割・解消したい文脈」のこと。

> 例: 便利さ・ステータス・美・共感 → 充足ニーズとの接続で価値が発生

## 5. 再生産と経済循環

一度の交換だけでは経済循環は生まれない。  
**差分を継続的に生み出す能力（再生産）**が必要。

```
Economy = Capital × Reproduction over Time
        = 差分の再生産システム
```

| 事例 | 差分の種類 | 評価圏 | 資本の種類 | 再生産の方法 |
|---|---|---|---|---|
| Apple | デザイン差分 | 市場・文化圏 | 経済資本＋文化資本 | 毎年の新製品 |
| 研究者 | 知識差分 | 学術界 | 文化資本 | 論文・引用の継続 |
| YouTuber | キャラクター差分 | SNS | 社会関係資本 | 動画の継続的公開 |

## 6. 所属と時間と評価圏の役割分担

| 軸 | 役割 | 特性 |
|---|---|---|
| **所属** | 中央値の定義（比較対象の決定） | 静的（主体の帰属セグメント） |
| **評価圏** | 差分の資本化（価値変換の場） | 外部的（主体の外に存在する） |
| **時間** | 資本の蓄積・差分の信号強度 | 動的（継続・再生産を生む） |

## 7. 価値判断の立場

**構造層として**、差分の良し悪しは評価圏によって決まり、経済論は内部的に定義しない。

```
本物の差分 vs 演出された差分 → 評価圏が価値を決定する
```

**目的層として**、Straは資本変換を「豊かさの実現」のために行う。  
どの評価圏に参入し、どの資本を蓄積するかの選択は、  
自身の豊かさの5条件（所属認識・更新可能性・選択肢・意志・実行基盤）に基づいて設計される。

> → 豊かさの5条件と評価圏選択の原則: [foundations/00-principles.md](00-principles.md)

> ⚠️ **倫理的留意**: 「評価圏が認めれば何でもよいのか」という問いに対しては、  
> 設計原則が回答する。他者の豊かさの5条件を損なう資本化はStraの原則に反する。  
> → [critique/01-vulnerabilities.md § 3-6](../critique/01-vulnerabilities.md)

## 8. 差分の消耗サイクル

価値ある差分は模倣される。模倣されると差分は平均化される。

```
差分の発生
  ↓
認識・評価圏への参入
  ↓
資本化・交換
  ↓
模倣・普及
  ↓
平均化（差分の消滅）
  ↓
新差分の探索（ループ）
```

## 10. 目標差分水準と象徴資本

### 目標差分水準（Target Deviation Level）

評価圏は差分を資本へ変換する場であるが、評価圏の内部には「何が価値ある差分と認識されるか」の閾値水準が存在する。これを**目標差分水準**と定義する。

| 概念 | 定義 | 決定因子 |
|---|---|---|
| **所属中央値** | 所属集団の実態分布から定まる参照点 | 集団の分布（恣意的に操作できない） |
| **目標差分水準** | 評価圏が認識・変換する差分の閾値水準 | 評価圏の権威構造（変動しうる） |

```
所属中央値からの設定差分
        ↓
    目標差分水準との照合
        ↓
   差分水準が閾値を超える → 評価圏内で認識・資本変換される
   差分水準が閾値未満   → 評価圏内で不可視のまま
```

目標差分水準は、評価圏ごとに「どの所属における・どの次元の・どの水準の差分が有効か」を規定する対応構造を持つ。この対応構造は評価圏に参入する主体の差分設計の指針となる。

### 象徴資本との関係

目標差分水準は固定された客観的基準ではなく、評価圏内で象徴資本を保有する主体の影響を受けて変動しうる。  
象徴資本保有者は目標差分水準の事実上の設定権を持つことで、変換効率の構造的非対称性を生み出す。

> → 象徴資本の定義・生成・作用: [foundations/08-symbolic-capital.md](08-symbolic-capital.md)

## 9. XRと体験経済

XRは知覚差分の生成装置であり、時間操作も可能にする。

```
XR × time → 体験経済の設計

例:
  未来情報の現在表示（ARナビ）
  過去の記憶の再現（XRアーカイブ）
  身体の知覚拡張（触覚フィードバック等）
```

XRが生成する知覚差分は文化圏・体験市場という評価圏に入ることで資本へ変換される。
