# 参照理論と比較考察

> **v0.5.0** | [← 因果グラフ](04-causal-graph.md) | [脆弱性 →](../critique/01-vulnerabilities.md)

ここで列挙する理論群は、本理論の正当性の根拠ではなく、  
思想的文脈を共有する参照点および比較対象として整理する。

---

## 1. 基盤となる理論

### 構造主義（Lévi-Strauss）

本理論の構造的出発点。  
意味は対象の本質に内在するのではなく、差異の体系の中での**位置関係**から生まれるという立場。

```
本質 → 意味（本質主義）
差異の体系内の位置 → 意味（構造主義）  ← 本理論の立脚点
```

交換もまた、対象の絶対的価値ではなく**体系内の差異構造**によって成立するとみる。  
本理論における所属・中央値・差分の体系はこの構造を継承している。

### 社会システム論（Luhmann）

Straに最も構造的に近い理論。

| Luhmannの概念 | Straとの対応 |
|---|---|
| 社会 = 差異のネットワーク | Identity = 差分の複合体 |
| 観察 → 区別 → 意味 | 主観中央値の設定 → 差分の観測 → アイデンティティ |
| 観察者が区別を作る | 主体が中央値を設定する |

## 2. 差分・差異に関する理論

### 差異の哲学（Deleuze）

差異それ自体（difference-in-itself）を存在論の中心に置く。

| | 差分 | 基準 | 目的 | タイプ |
|---|---|---|---|---|
| **Stra** | 基準からの差分 | 基準が先にある | 測定と調整 | 工学的 |
| **差異の哲学** | 差分が先にある | 差分から基準が生まれる | 生成と流動 | 形而上学的 |

本理論とは差分の順序が逆であり、立場は異なる。  
ただしドゥルーズはXR・知覚・イメージの哲学化にも言及しており、知覚差分拡張の議論とは共鳴しうる。

### 情報の差分論（Bateson）

```
"Information = difference that makes a difference"
（意味を持つ差分が情報である）
```

差分がシグナルになる条件（持続・充足ニーズとの接続）を考察する本理論と構造が近い。  
Human = information node という拡張も考察の対象となる。

### 情報理論（Shannon）

情報量 = 不確実性の削減。  
差分の希少性・予測困難性が価値を持つという考察と接続する。  
差分が「ノイズ」か「シグナル」かを判断する時間論的議論とも関連する。

## 3. フィードバック・制御に関する理論

### サイバネティクス（Wiener）

基準値 → 逸脱の観測 → フィードバック → 補正というループ構造。  
本理論における「中央値の仮置き → 外部フィードバックによる補正」プロセスと同型。

### ベイズ推定

事前確率（中央値の仮置き）を観測によって更新するプロセス。  
参照基準の動的補正モデルと対応する。

---

## 4. 価値・資本・市場に関する理論

### 資本理論（Bourdieu）

本理論の経済論の基盤となる理論。  
資本を経済資本・社会関係資本・文化資本の3種に分類し、それぞれが異なる社会的場（champ）で機能することを示す。

| Bourdieuの概念 | Straとの対応 |
|---|---|
| 社会的場（champ） | 評価圏（差分が評価される社会環境） |
| 資本の3分類 | 差分が変換される資本の3種 |
| ハビトゥス（habitus） | 中央値への適応として生じる行動傾向（差分生成の素地） |

Straにおける「評価圏 → 資本化」のプロセスは、ブルデューの「場における資本変換」と構造的に対応する。

### ジョブ理論（Christensen）

人はプロダクトを「ある文脈を解消するために雇う（hire）」という立場。  
差分が充足ニーズ（Job）と接続されるとき価値になるという本理論の経済論と近い。  
本理論では「欲望」の代わりに「充足ニーズ（Job）」という語を採用している。

### ポジショニング理論（Trout & Ries）

ブランドの価値は競合との相対的差異によって決まる。  
差分の市場的意味を扱う点で接続できる。

### 参照集団理論（Merton）

人は絶対的基準ではなく、自身が参照する集団を基準に自己を評価する。  
所属ごとに中央値が異なるという本理論の認識論的根拠として接続できる。

### 差別化戦略（Porter）

競争優位は本質的優位ではなく他との差異によって生まれる。  
差分の経済化モデルと構造的に近い。

## 5. 理論の交差点

本理論は以下の3つの系譜の交差点に位置する。

```
構造主義（Lévi-Strauss → Luhmann）
  ×
サイバネティクス（Wiener → ベイズ推定）
  ×
資本理論（Bourdieu）
  ↓
Stra
```

## 6. AI時代との接続

### AI = 中央値の形成への主要な寄与因子

AIは統計的パターンから平均的な出力を量産し、中央値の形成に強く寄与する。  
中央値は社会・制度・市場など複数の力によって形成されるが、AIの量産的出力はその参照点として急速にウェイトを増している。  
AIが中央値の形成に最も強く寄与する世界では:

```
人間の価値 = AIが寄与する中央値からの逸脱量（差分）
```

### 人間固有の差分

AI がより精度高く平均的な出力を量産できるようになるほど、人間に固有の差分とは何かという問いが派生する。  
現時点での仮説として、以下の4要因が考察されている。

| 要因 | 内容 |
|---|---|
| **有限性（死）** | 一回性・不可逆性が選択にコストと意味を付与する |
| **身体** | 身体 × 環境から生まれる経験。記号処理であるAIには持ちえない意味の源泉 |
| **偶然** | AIは確率で動くが、人間は「偶然 × 解釈」で動く |
| **意図的な時間設計** | AIの目的は外部から与えられるが、人間は時間のストーリーを自律的に設計できる |

これらは未検証の仮説であり、理論的精緻化は今後の課題として残る。

## 6. 対立する思想的立場

| 対立概念 | 内容 | Straの立場 |
|---|---|---|
| **本質主義** | 人・組織には固定本質がある | 本質ではなく差分がアイデンティティを作る |
| **普遍主義** | 普遍的な価値・善悪尺度がある | 価値は市場・目的系により決まる（相対主義） |
| **ロマン主義的真正性** | 本物のオリジナリティに価値がある | 演出差分も市場が認めれば価値になる |
| **人間中心主義** | 人間は機械に還元できない特権的本質を持つ | 人間・組織・プロダクトを差分構造として並列に扱う（ポストヒューマン寄り） |
