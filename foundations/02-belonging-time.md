## 1. 所属（Belonging）とは

所属とは、対象（個人・組織・プロダクト）が参照する**複数のセグメント**のことを指す。  
各所属に対して固有の中央値が存在し、対象はその中央値からの差分を持つ。

アイデンティティは単一の所属から決まるのではなく、**複数所属における差分の複合値**として形成される。

```
Identity space = multiple belonging sets
Identity = Σ deviation(belonging_i)
```

---

## 2. 所属の分類

### 2-1. 絶対的所属
物理的・生物的に変えにくい条件を基準とする所属。

| 例 | 特徴 |
|---|---|
| 年齢・世代 | 変えられない |
| 身体・健康状態 | 生物的制約 |
| 国籍・出生地 | 原則変更困難 |
| 生物的条件（性別等） | 根本的に固定 |

### 2-2. 相対的所属
前提となる指標が相対的であり、社会的影響度が高く変化しうる所属。

| 例 | 特徴 |
|---|---|
| 職業・役職 | 変えられる |
| 文化・趣味 | 文脈依存 |
| 思想・価値観 | 流動的 |
| 市場ポジション | 競争によって変化 |

## 3. 所属の重み（Weight）

複数の所属は等価ではなく、対象ごとに**重要度（weight）**が異なる。  
このweightが変わると、アイデンティティも変わる。

```
例:
ある個人の所属と重み
  国籍      weight 0.1
  職業      weight 0.5
  趣味      weight 0.2
  思想      weight 0.2

Weighted Identity = Σ (deviation_i × weight_i)
```

## 4. 中央値（Reference）の性質

中央値は以下の性質を持つ:

1. **主観的推定** — 所属集団の分布中心を、対象（主体）が推定した値を起点とする。中央値は集団の実態分布によって規定され、主体が恣意的に設定するものではない
2. **定量的参照の許容** — 外部の統計・データを参照することは問題ない
3. **外部フィードバックによる補正** — オーディエンスからの相対的評価で随時修正
4. **動的** — 時間とともに社会の平均自体が変化するため、基準も変容する

```
中央値推定プロセス:
  主観的仮置き → 外部フィードバック観測 → 差分の認識 → 推定値の修正
```

このサイクルは **サイバネティック・ループ**（制御理論）・**ベイズ更新**と同構造。

> ⚠️ **所属中央値と目標差分水準の区別**  
> 所属中央値は集団の分布から定まる「設定差分」の算出基準である。  
> 評価圏が「価値ある差分」と認識する閾値水準（**目標差分水準**）とは異なる概念として区別される。

## 5. 時間（Time）

### 5-1. 時間の3つの役割

| 役割 | 説明 |
|---|---|
| **①差分の持続** | 瞬間的差分はノイズ。継続する差分がアイデンティティになる |
| **②信号強度（信頼度）** | 短期差分 = ノイズ候補。長期差分 = シグナル（信頼性が高い） |
| **③差分の生成** | 中央値（社会平均）自体が時間によって変化し、差分も変化する |

```
Deviation = f(reference(t))
時間によって参照基準が変わり、それに伴い差分も変わる
```

### 5-2. 時間の3層モデル（Layered Time）

```
短期：瞬間差分
  → 行動・投稿・ファッション・発言

中期：行動パターン差分
  → キャリア・趣味・生活習慣・プロダクトのコンセプト

長期：存在構造差分
  → 価値観・思想・人格・ブランドの核
```

完全な定式:

```
Identity = Σ deviation(belonging_i × time_layer_j)
```

### 5-3. 時間 × 有限性

人間に固有な要素として、「死（有限性）」が時間に意味を与える。

```
Identity = Σ deviation × persistence × finitude

有限時間で持続した差分 = 人間的意味の核
```

## 6. XRとの接続

XR（Extended Reality）は、この理論における**知覚中央値を変える技術**として位置づけられる。

```
通常:    空間認識 → 目（知覚）
XR:      空間 + デジタルレイヤー → 知覚差分の生成

ARナビ = 未来情報を現在に表示 = 時間差分の圧縮

XR = perception deviation engine（知覚差分生成装置）
XR = 知覚差分 + 時間操作
```
