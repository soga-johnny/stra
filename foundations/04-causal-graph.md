# 因果グラフ — 入力差分と出力差分

> **v0.5.0** | [← 経済論](03-economics.md) | [参照理論 →](05-references.md)

---

## 1. なぜ因果グラフが必要か

Straの基本フローは「差分 → 評価圏 → 資本」という変換を記述するが、  
それだけでは「現在の状態から目標の状態へどう動くか」という**因果的・操作的な問い**に答えられない。

そこで「入力差分」と「出力差分」を区別し、その因果関係を定義する。

---

## 2. 入力差分と出力差分の定義

### 入力差分（Input Deviation）

主体が**意図的に生み出す**差分。行動・投資・選択によって形成される。

> 主体のコントロール下にある差分。

| 例 | 内容 |
|---|---|
| スキル習得 | 専門技能の獲得 → 職業所属内の中央値からの逸脱 |
| 時間投資 | 特定分野への集中的な時間配分 |
| 関係設計 | ネットワークの構築・所属の意図的な変更 |
| ポジショニング | 評価圏の選択・発信文脈の設計 |

### 出力差分（Output Deviation）

入力差分が評価圏を通過した結果として**観測される**差分。  
市場・他者・コミュニティによる認識・評価の変化として現れる。

> 評価圏による変換の結果として生じる差分。主体がコントロールできない部分を含む。

| 例 | 内容 |
|---|---|
| 市場評価の変化 | 報酬・受注・採用機会の増減 |
| 資本の変化 | 経済資本・社会関係資本・文化資本の増減 |
| ポジションの変化 | 評価圏内での相対位置の移動 |

### 両者の関係

```
入力差分 → [評価圏フィルター] → 出力差分
```

同じ入力差分でも、評価圏が異なれば出力差分は変わる。  
評価圏との適合度が、変換効率を左右する。

## 3. 因果グラフ

```
目標出力差分の設定（ゴールの定義）
          ↓
評価圏の特定・選択
          ↓
評価圏の目標差分水準の観測（その評価圏で認識・変換される差分の閾値）
          ↓
現在の設定差分の測定（各所属における所属中央値からの現在値）
          ↓
差分ギャップの算出（目標差分水準 − 設定差分 = 埋めるべきギャップ）
          ↓
入力差分の設計（行動・投資・選択の計画）
          ↓
入力差分の実行
          ↓
出力差分の観測（評価圏での反応・資本変化）
          ↓
フィードバック
  ├─ 所属中央値の推定補正（参照集団の見直し・推定精度の修正）
  ├─ 目標差分水準の見直し（評価圏の閾値の読み違いがあった場合）
  ├─ 評価圏の見直し（場の選択が最適でなかった場合）
  └─ 入力差分の再設計（行動計画の修正）
          ↓
          └──────────── ループ
```

## 4. 逆算設計（ゴールからの設計）

本理論では、目標を「望ましい出力差分の状態」として定義することで、  
逆算的に必要な入力差分を設計できる。

```
Step 1: 目標出力差分を定義する
        「年収1000万円」= 特定の評価圏（市場）における
                        一定水準の経済資本への変換

Step 2: その出力差分を生む評価圏を特定する
        → どの市場・業界・コミュニティで評価されるか

Step 3: 評価圏の目標差分水準を観測する
        → 年収1000万円帯が参入する評価圏で認識される差分のパターン（閾値水準）

Step 4: 自身の設定差分を測定する
        → 各所属における所属中央値からの現在値（設定差分）

Step 5: 差分ギャップを可視化する
        → 目標差分水準 − 設定差分 = 差分ギャップ（入力差分で埋めるべき量）

Step 6: 入力差分を設計・実行する
        → スキル・時間・関係・ポジションの投資計画
```

---

## 5. 理論上の前提と留意点

### 理論が対応できる範囲

- 目標を差分として記述すること（定義）
- 評価圏と中央値の観測（現状把握）
- 入力差分の設計と優先順位付け（計画）
- 出力差分の観測と補正（フィードバック）

### 理論が対応しない範囲

- 入力差分を実行する動機・継続力の生成
  （これは有限性・身体・感情という人間固有の差分源泉に関わる領域であり、本理論の記述対象外）
- 評価圏自体の価値判断（どの評価圏が「良い」かは本理論では定義しない）

---

## 6. 入力差分と資本の対応

| 入力差分の種類 | 主な変換先評価圏 | 生成される資本 |
|---|---|---|
| 専門スキルの習得 | 職業市場・学術界 | 経済資本・文化資本 |
| ネットワーク形成 | コミュニティ・業界 | 社会関係資本 |
| 発信・可視化 | SNS・メディア | 社会関係資本・文化資本 |
| 実績の蓄積 | 市場・顧客 | 経済資本 |
| 所属の変更 | 新しい評価圏への参入 | 複合資本 |
