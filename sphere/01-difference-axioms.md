## 1. 中心命題

個人・組織・プロダクトなどの主体は、固定した本質ではなく、  
**複数の所属における中央値からの差分の総体**として記述できる。

この差分は社会的環境（評価圏）の中で評価され、  
時間を通して持続・再生産されることで**資本へと変換される**。

---

## 2. 基本フロー

```
主体 → 所属 → 中央値 → 差分 → 評価圏 → 資本 → 時間 → 再生産
```

---

## 3. 基本定式

```
Identity = Σ (所属ごとの差分ベクトル × 持続時間 × 有限性)

Capital  = Difference × EvaluationSphere × Time

Economy  = Capital の再生産システム
```

より簡潔に書くと:

```
I = Σ deviation(belonging_i × time_layer)
C = deviation × sphere × t
```

## 4. 6つの理論的柱

### 4-1. 存在論
個人・組織・プロダクトは固定本質ではなく、複数所属における差分の総体として存在する。  
→ 非本質主義・構造主義的立場（Lévi-Strauss の構造主義を起点とする）

### 4-2. 認識論
差分を測る基準（中央値）は客観真理ではなく、主体が仮設するものである。  
外部からのフィードバックを通じて補正される**動的基準**である。

### 4-3. 時間論
差分は瞬間的に存在するだけでは「ノイズ」に過ぎない。  
持続・反復・変容の中で「シグナル」となり、意味（アイデンティティ）を獲得する。  
時間は差分の**信号強度と耐久性**を評価する軸。

### 4-4. 社会論（評価圏）
差分が社会的価値を持つかどうかは**評価圏**によって決まる。  
評価圏とは差分が評価され資本へ変換される社会的環境であり、所属とは異なる概念である。  
同一の差分でも評価圏が異なれば価値は変化する。

### 4-5. 経済論（資本化）
差分は評価圏での認識・充足ニーズ（Job）との接続・交換・再生産を経て資本へ変換される。  
資本はブルデューの分類に従い、経済資本・社会関係資本・文化資本の3種に対応する。

### 4-6. 技術論
テクノロジー（特にXR）は知覚と時間の差分を設計・拡張しうる**差分生成装置**である。

## 5. 世界観の構造

| 主体 | Stra上の定義 |
|---|---|
| 世界 | 平均構造 |
| 社会 | 評価圏の集合 |
| 人間 / 組織 | 差分構造（平均からの逸脱） |
| テクノロジー（XR等） | 差分生成・拡張装置 |
| AI | 中央値の形成への主要な寄与因子 |
| **人間の価値** | **AIが強く寄与する中央値からの逸脱量** |

---

## 6. 価値判断の立場

**構造層として**、差分に本質的な善悪はない。  
差分は**まず分類**であり、価値は評価圏・市場・目的系・文脈によって後から付与される。

> 例外: ある目的を持つシステム（ゲームデバイス・経済的ゴールを持つプロダクト等）では、  
> そのゴールに忠実な差分が「良い差分」として定義されうる。

**目的層として**、Straはこの価値中立な構造論を「豊かさの実現」という規範的目的のために使う。  
差分を設計する意図・何のために資本化するかという問いは、構造論が答えるのではなく、  
設計原則が答える。

> → 豊かさの定義・5条件・倫理的含意: [locus/00-principles.md](../locus/00-principles.md)

## 7. 人間固有の差分

AIがあらゆる差分を模倣・生成できるようになった場合でも、  
以下の4要素は人間に固有の差分源泉として残ると考えられる（いずれも仮説）。

| 要素 | 内容 |
|---|---|
| **有限性（死）** | 一回性・不可逆性が意味を生む。AIは繰り返し可能だが人間の決断は不可逆 |
| **身体** | 身体 × 環境の経験。寒さ・痛み・恋愛などの身体状態に基づく意味形成 |
| **偶然** | 確率ではなく「偶然 × 解釈」で動く。たまたまの出会い・経験による人生の分岐 |
| **意図的な時間設計** | 「10年後こうなりたい」という時間のストーリー設計。AIの目的は外部から与えられるが、人間は自ら設計する |
