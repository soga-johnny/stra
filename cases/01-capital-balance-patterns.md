## 1. このドキュメントの位置づけ

組織主体の豊かさを記述するとき、「どの評価圏でどの資本を蓄積・変換しているか」という資本構成の傾向は、差分設計の方向性を読む重要な補助線となる。

本ドキュメントは、組織形態・ビジネスモデル・産業セグメントごとの**資本構成傾向を抽象的なパターン**として整理した種（seed）ドキュメントである。  
特定の企業・団体・個人名は記載しない。具体的な個別事例・定量的データへの拡張は後続作業として予定する。

> 参照: ブルデューの資本構成概念（Volume / Composition）→ [foundations/05-references.md](../foundations/05-references.md)

---

## 2. 記述の凡例

| 記号 | 資本の種類 | 内容 |
|---|---|---|
| **Ec** | 経済資本 | 収益力・財務資源・物的資産 |
| **Sc** | 社会関係資本 | 顧客基盤・ネットワーク・信用・ブランド認知 |
| **Cc** | 文化資本 | 専門知識・技術・ブランド意味・知的財産・ノウハウ |

| 記号 | 意味 |
|---|---|
| ◎ | 優位（当該組織の主要差分源泉・競争優位の核） |
| ○ | 中位（一定水準を保有） |
| △ | 劣位（相対的に薄い・外部依存） |

## 3. ビジネスモデル別パターン

### パターン A — 大量生産・規模型

**資本構成傾向**: Ec ◎ / Sc ○ / Cc △

```
主要評価圏:   市場（コスト・価格競争）
主要差分:     生産規模・コスト効率・流通網の広さ
変換軌跡:     Ec → Ec（自己再生産）
```

- 経済資本（設備・規模・物流）が主要な差分源泉
- 文化資本が低水準になりやすく、差分の平均化速度が速い
- 模倣耐性が低いため、規模の維持そのものが差分の防衛戦略になる
- **組織的適応状態の傾向**: 固体〜液体（スケールが硬直を生みやすい）
- **リスク**: 新規参入者のスケール獲得・自動化進展で差分が急速に消耗する

---

### パターン B — プレミアム・ブランド型

**資本構成傾向**: Ec ○〜◎ / Sc ○ / Cc ◎

```
主要評価圏:   文化圏・ステータス評価圏・プレミアム市場
主要差分:     ブランド意味・審美的差分・ストーリーの独自性
変換軌跡:     Cc → Ec（文化的意味 → 価格差分 → 経済資本）
```

- 文化資本が高い模倣障壁を生む（希少性の構造的確保）
- 同じ機能水準でも Cc の差分によって価格差分が正当化される
- Cc の一貫性（時間軸での持続）がアイデンティティの核になる
- **組織的適応状態の傾向**: 気体〜プラズマ（ブランドの核を守りながら評価圏を渡り歩く）
- **リスク**: Cc の意味が陳腐化すると価格差分の根拠が崩れる

---

### パターン C — プラットフォーム型

**資本構成傾向**: Sc ◎ / Ec ○〜◎ / Cc △〜○

```
主要評価圏:   プラットフォーム上の参加者コミュニティ・広告市場・データ市場
主要差分:     ネットワーク効果（参加者数・接続密度）
変換軌跡:     Sc → Ec（ネットワーク価値 → 広告・取引手数料・データ収益）
```

- 初期は赤字でも Sc を積み上げ、後から Ec に変換するモデル
- Sc の独占化が最大の差分（参加者の乗り換えコストが参入障壁）
- Cc が低いため代替プラットフォームへの移行リスクを Sc で封じる構造
- **組織的適応状態の傾向**: プラズマになると評価圏を自ら形成できる（新規評価圏の創出）
- **リスク**: Sc は急激な流出が起きうる。規制・分散化技術による Sc の侵食

---

### パターン D — 専門知識集約型サービス

**資本構成傾向**: Cc ◎ / Sc ○〜◎ / Ec △〜○

```
主要評価圏:   専門職コミュニティ・特定産業のクライアント市場
主要差分:     知識の深度・ノウハウの希少性・問題解決の実績
変換軌跡:     Cc → Sc → Ec（知識 → 信頼・紹介 → 収益）
```

- 知識・技術・ノウハウが主要な差分源泉
- Sc（顧客ネットワーク・紹介経路・業界信用）が変換の経路として機能する
- 属人性が高いため、組織としての Cc 蓄積が難しい
- **組織的適応状態の傾向**: 液体〜気体（属人依存が強いと組織としての適応が遅れる）
- **リスク**: 知識の陳腐化（AI による Cc の平均化）・主要人材の離脱

---

### パターン E — コンテンツ・メディア型

**資本構成傾向**: Cc ◎ / Sc ◎ / Ec △〜○

```
主要評価圏:   認知圏（SNS・メディア・配信プラットフォーム）・広告市場・ライセンス市場
主要差分:     コンテンツの独自性・オーディエンスとの関係性の深度
変換軌跡:     Cc → Sc → Ec（コンテンツ → オーディエンス蓄積 → 広告・課金）
```

- 独自コンテンツが文化資本、オーディエンスが社会関係資本
- Ec 変換効率はプラットフォームの条件・アルゴリズムに強く依存する
- Sc の揮発性が高い（注目は集まりやすく消えやすい）
- **組織的適応状態の傾向**: オーディエンスとのフィードバックループが機能するとプラズマへ近づく
- **リスク**: Cc の消耗速度が速い。Sc の分散化（プラットフォーム依存）

---

### パターン F — 社会的組織型（NPO・公共機関・協同組合）

**資本構成傾向**: Sc ◎ / Cc ○ / Ec △

```
主要評価圏:   受益者コミュニティ・支援者圏・社会的正当性の場
主要差分:     使命の独自性・受益者との関係性・社会的信頼の厚さ
変換軌跡:     Sc ⇄ Cc（相互強化）→ Ec 調達（補助金・寄付・会費）
```

- 使命・信頼・コミュニティが Sc と Cc の源泉
- Ec は外部調達（補助金・寄付・会費）依存であり、Sc・Cc が Ec 調達の条件になる
- ミッション固着による固体化リスクがある一方、コミュニティとのループが機能するとプラズマになれる
- **組織的適応状態の傾向**: 固体か、またはプラズマの二極になりやすい
- **リスク**: Ec 依存先の喪失（資金源の変化）・ミッションの形骸化

---

### パターン G — 垂直統合型

**資本構成傾向**: Ec ◎ / Cc ◎ / Sc ○

```
主要評価圏:   市場（複数層）＋文化圏（ブランド）
主要差分:     バリューチェーン全体の差分設計能力・品質一貫性
変換軌跡:     Ec + Cc の同時蓄積（各層での資本収穫）
```

- 生産から流通・販売まで自社で差分を設計できるため、Cc と Ec が同時に高い
- 各バリューチェーン層で異なる評価圏から資本を収穫できる
- 組織規模拡大と固体化リスクのトレードオフが常に存在する
- **組織的適応状態の傾向**: 気体〜プラズマになりやすいが、組織規模が大きくなると固体化圧力も高い
- **リスク**: 各層のスペシャリストとの競合・サプライチェーンの変動

---

### パターン H — 初期スタートアップ型

**資本構成傾向**: Cc ○〜◎ / Sc △〜○ / Ec △

```
主要評価圏:   投資家・初期ユーザーコミュニティ・業界メディア
主要差分:     新技術・新コンセプト・未解決課題への独自アプローチ
変換軌跡:     Cc → Sc（認知・ネットワーク構築）→ Ec（収益化・調達）
```

- 資本総量は低いが、Cc の差分（新技術・新コンセプト）が競争優位の核
- Sc と Ec は軌跡として後から変換するロードマップが基本
- 個人の適応状態が組織の適応状態に直結する（小規模のため）
- **組織的適応状態の傾向**: 創業チームのプラズミズム状態に依存する
- **リスク**: Cc の希少性が失われる前に変換軌跡を完成させられるかどうか

## 4. 資本変換の軌跡パターン（横断整理）

| 変換軌跡 | 該当するパターン |
|---|---|
| Cc → Ec | B（プレミアムブランド）、D（専門知識集約） |
| Sc → Ec | C（プラットフォーム）、E（コンテンツ・メディア） |
| Cc → Sc → Ec | D（専門知識）、E（コンテンツ）、H（スタートアップ） |
| Ec → Ec（自己再生産） | A（大量生産・規模） |
| Ec + Cc（同時蓄積） | G（垂直統合） |
| Sc ⇄ Cc（相互強化） | F（社会的組織） |
| Cc → Ec（Cc → 価格差分） | B（プレミアムブランド） |

## 5. 資本構成とプラズミズム状態の対応（仮）

資本構成の傾向と組織的適応状態（[foundations/06-plasmism.md § 9](../foundations/06-plasmism.md)）には以下の関係が仮定される。

| 組織的適応状態 | 資本構成の傾向 |
|---|---|
| **固体** | Ec 単一依存（または Cc の陳腐化）。評価圏が単一に固着 |
| **液体** | Ec + Sc の基本構成。複数評価圏を認識しているが変換は限定的 |
| **気体** | Cc の蓄積が始まる。複数評価圏を意図的に設計している |
| **プラズマ** | Ec / Sc / Cc のバランスが高水準。資本変換を自律的にサイクルできている |
