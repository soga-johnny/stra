## 概要

Straはテクノロジーと資本が流動する世界において、  
主体が**豊かさ**（所属認識・更新可能性・選択肢・意志・実行基盤の同時成立）を持ちながら資本と接続するための  
オープンスタンダードな実践的思想体系である。

存在論・社会理論・経済理論・技術思想の交差領域に位置し、  
中心的な問いは「**あるものが、あるものとして認識されるとはどういうことか**」である。  
これを「本質の発見」ではなく「参照基準からの差分の記述」として答え、  
その差分の設計と運用を通じて主体の豊かさを実現することを目的とする。

> Straの設計原則:  
> → [locus/00-principles.md](locus/00-principles.md)

一文で言えば、

> **主体は複数の所属における差分として存在し、  
> その差分は評価圏において資本へ変換され、  
> 通貨として追跡・運用されながら時間の中で再生産される。**

「人間とは何か」という存在論的問いにとどまらず、  
「差分をどう測るか」「どう設計・運用するか」まで踏み込む点で、  
Straは**思想から始まり、実装可能な実践フレームワーク**としての運用可能性に特に着目している。

### 基本フロー

```
主体 → 所属 → 中央値 → 差分 → 評価圏 → 資本 → 時間 → 再生産
                                  ↑
                             通貨
```

### 実践への拡張

差分は「記述」するだけでなく、**設計・操作**することができる。  
主体が意図的に生み出す**入力差分**と、評価圏を通じて観測される**出力差分**を区別し、  
その因果関係を設計することで、目標状態への逆算的なアプローチが可能になる。

```
目標出力差分の設定 → 評価圏の特定 → 目標差分水準の観測 → 現在の設定差分の測定
  → 差分の算出 → 入力差分の設計・実行 → 出力差分の観測 → 補正ループ
```

→ 詳細: [locus/04-deviation-design.md](locus/04-deviation-design.md)

## 構成要素

| 要素 | 説明 |
|---|---|
| **対象** | 個人・組織・プロダクト・ブランド・思想など。広い単位に適用できる |
| **所属** | 対象が複合的に帰属する複数のセグメント。ひとつの対象が同時に複数の所属を持っている状態が前提となる。**絶対的所属**（例：年齢・国籍・身体条件など、原則変えられないもの）と**相対的所属**（例：職業・文化圏・市場ポジション・思想など、文脈によって変わりうるもの）に分かれる |
| **中央値** | 各所属の中で「一般的・標準的とみなされる状態」を指す参照点。絶対的な基準ではなく、主体と文脈によって異なる |
| **差分** | 対象が中央値からズレている量と方向。このズレの複合体がアイデンティティになる |
| **評価圏** | 差分が評価され資本へ変換される社会的環境。所属とは異なる概念で、同一の差分でも評価圏によって価値は変化する（例：市場・文化圏・学術界・SNS・芸術界など） |
| **資本** | 評価圏において認識された差分が変換されたもの。ブルデューの資本理論と接続し、**経済資本**（市場評価）・**社会関係資本**（ネットワーク評価）・**文化資本**（文化評価）の3種に分類できる |
| **時間** | 差分の持続性を評価する軸。**瞬間的差分**（例：一度だけの奇抜な発言・単発の施策）はノイズとして扱われやすく、**持続的差分**（例：10年間一貫したデザイン哲学・長期にわたる専門性）がシグナル＝アイデンティティとなる |
| **通貨** | 差体が投じた入力差分と評価圏から回収した出力差分の流量を追跡する主観的単位。複式に記述され、主要用途は**新しい評価圏へのアクセスコスト**。変換レートは所属・評価圏・プラズミズム状態・時間の4変数で変動する |

## ドキュメントの3層構造

Straのドキュメントは思想体系の構造に対応した3層で構成される。

| 層 | フォルダ | 日本語概念名 | 役割 |
|---|---|---|---|
| **圏観** | `sphere/` | 圏観 | 差分構造の客観的記述体系。評価圏・資本変換・象徴資本など「社会がどう機能しているか」の定義層 |
| **差体** | `locus/` | 差体 | 差分空間における主体の座標体系。豊かさ・プラズミズム・設計・日次実行など「主体がどう在るか」の方法論層 |
| **通貨** | `currency/` | 通貨 | 差体の実行エネルギーの循環層。入力差分の投入と出力差分の回収を複式モデルで追跡し、評価圏アクセスへ運用する |

```
sphere/ ─── 差分構造・評価圏・資本変換の法則（客観）
               ↕
locus/ ──── 差体の目標設定・実行・適応状態（主体）
               ↕
currency/ ── 差分流量の追跡・変換・評価圏アクセスへの運用（循環）
```

## 理論の強み

**① 実装可能性が高い**  
多くの思想は存在論的問いで終わる。本理論は差分の測定・調整・運用まで進める。  
ダッシュボード・ブランド戦略・組織設計・プロダクト設計への展開が可能。

**② 主観と外部評価を両立する**  
中央値は主体が設定するが、外部フィードバックで随時補正される。  
自己定義だけでも他者評価だけでも成立しない構造は、現実の組織・個人の動態に即している。

**③ 複数所属を前提とする**  
人は単一の本質ではなく、複数の文脈にまたがって存在する。  
個人・組織・プロダクトはいずれも複数市場・複数役割・複数規範の交点にある。

**④ 経済・資本と接続できる**  
差分を哲学概念にとどめず、評価圏・充足ニーズ（Job）・資本・再生産のメカニズムに接続する。  
経済資本・社会関係資本・文化資本との接続により、差分がどの文脈でどの種類の資本へ変換されるかを記述できる。

**⑤ 実行エネルギーを定量的に記述できる**  
差体の日々の実行を通貨の複式モデルで追跡することで、「どの評価圏への投資が効率的か」「次にアクセスすべき評価圏はどこか」を定量的に判断できる。哲学的定義と日次実行の間の橋渡しが通貨層によって可能になる。

## テクノロジーとの親和性

### AI との接続

AI は統計的パターンから平均的な出力を高精度・高速で量産し、中央値の形成に強く寄与する。  
本理論の枠組みでは、AI は「中央値の形成への主要な寄与因子」として役割付けられる。

> 中央値は社会・制度・市場・文化など複数の力によって形成されるが、AIが量産する平均的出力はその参照点としてのウェイトを急速に高めている。

```
AI が中央値の形成に最も強く寄与する世界では、
差分（= その中央値からの逸脱）が、人間・組織のアイデンティティを記述する指標のひとつとなりうる。
```

同時に、AI が中央値の形成に強く寄与する状況は、**人間に固有の差分とは何か**という問いを派生させる。  
現時点での仮説として、以下の4要因が人間特有の差分源泉として考察されている。

| 要因 | 内容 |
|---|---|
| **有限性（死）** | 人生の一回性・不可逆性が選択にコストと意味を付与する。AI の計算は繰り返し可能だが、人間の決断は不可逆 |
| **身体** | 寒さ・痛み・疲労・感情など、身体 × 環境から生まれる経験。記号処理である AI には持ちえない意味の源泉 |
| **偶然** | AI は確率で動くが、人間は「偶然 × 解釈」で動く。予期せぬ出会いや経験が人生の分岐を生む |
| **意図的な時間設計** | AI の目的は外部から与えられるが、人間は時間のストーリーを自律的に設計できる |

これらは未検証の仮説であり、理論的精緻化は今後の課題として残る。

### XR との接続

XRは、本理論における**知覚中央値を操作する技術**として位置づけられる。  
物理空間にデジタルレイヤーを重ねることで、知覚の基準点そのものを変位させる効果を持つ。

```
XR = 知覚差分の生成装置
XR × 時間操作（例：AR ナビによる未来情報の現在表示）= 時間差分の圧縮
```

身体知覚を拡張するという XR の特性は、上記「身体」要因とも接続する論点となる。

## 参照理論

本理論の立脚点・比較対象・接続可能な理論群を以下に整理する。  
いずれも「本理論の正当性の根拠」ではなく、思想的文脈を共有する参照点として列挙する。

**基盤となる理論**

| 理論                     | 参照点                                             |
| ---------------------- | ----------------------------------------------- |
| **構造主義**（Lévi-Strauss） | 意味は対象の本質ではなく、差異の体系の中の位置関係から生まれるという立場。本理論の構造的出発点 |
| **社会システム論**（Luhmann）   | 社会を差異のネットワークとして記述。「観察 → 区別 → 意味」の構造が本理論の認識論と近い  |

**差分・差異に関する理論**

| 理論                  | 参照点                                                                                          |
| ------------------- | -------------------------------------------------------------------------------------------- |
| **差異の哲学**（Deleuze）  | 差異それ自体を存在論の中心に置く。本理論とは「差分の順序」が逆（差分が先 vs 基準が先）だが、差分を本質より優先する点で共鳴する                            |
| **情報の差分論**（Bateson） | "information = difference that makes a difference"（意味を持つ差分が情報である）。差分がシグナルになる条件を考察する本理論と構造が近い |
| **情報理論**（Shannon）   | 情報量 = 不確実性の削減。差分の希少性・予測困難性が価値を持つという考察と接続する                                                   |

**フィードバック・制御に関する理論**

| 理論                   | 参照点                                                     |
| -------------------- | ------------------------------------------------------- |
| **サイバネティクス**（Wiener） | 基準値 → 逸脱の観測 → フィードバック → 補正というループ構造。本理論における中央値の補正プロセスと同型 |
| **ベイズ推定**            | 事前確率（中央値の仮置き）を観測によって更新するプロセス。参照基準の動的補正モデルと対応する          |

**価値・資本・市場に関する理論**

| 理論                          | 参照点                                                            |
| --------------------------- | -------------------------------------------------------------- |
| **資本理論**（Bourdieu）          | 資本を経済資本・社会関係資本・文化資本の3種に分類。差分が評価圏を通じていずれかの資本に変換されるという本理論の経済論の基盤 |
| **ジョブ理論**（Christensen）      | 人はプロダクトを「ある文脈を解消するために雇う」。差分が充足ニーズ（Job）と接続されるとき価値になるという経済論と近い   |
| **ポジショニング理論**（Trout & Ries） | ブランドの価値は競合との相対的差異によって決まる。差分の市場的意味を扱う点で接続できる                    |
| **参照集団理論**（Merton）          | 人は絶対的基準ではなく、自身が参照する集団を基準に自己評価する。所属ごとの中央値設定の認識論的根拠              |
| **差別化戦略**（Porter）           | 競争優位は本質的優位ではなく他との差異によって生まれる。差分の経済化モデルと構造的に近い                   |

→ 詳細比較: [sphere/05-references.md](sphere/05-references.md)
